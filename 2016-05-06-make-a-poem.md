``` html
《立夏》  
    —— 胡写的
蚯蚓出 /  
蝼蝈鸣 /  
哦 /  
夏日已来到 /  

若未睡 /  
起来嗨 /  
听 /  
蛙与蛐在笑 /  
```
<!--more-->

PS:  
解释一下，季节入夏了，睡的比较晚的同学也会注意到半晚的时候外面草丛里青蛙与蛐蛐的叫声是最亮的，笑是我对它们乱叫的拟人，用笑我有两层含义如下：1，欢笑，夏天终于来了，好开心以至于睡不着觉；2，嘲笑，蛙和蛐蛐在嘲笑我们这群人（程序猿），那么晚还不睡有点孬，所以添油加醋使劲叫… ^^
